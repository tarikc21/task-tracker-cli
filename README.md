# Task Tracker CLI

Simple CLI task tracker written in C# using .NET.

## Features

- Add tasks
- List tasks
- Mark tasks as done
- Delete tasks
- Stores tasks in JSON file

## Commands

Add task

dotnet run add "Task description"

List tasks

dotnet run list

Mark task as done

dotnet run done 1

Delete task

dotnet run delete 1

## Example

dotnet run add "Learn C#"

dotnet run list

1 | Learn C# | Todo

## Technologies

- C#
- .NET
- JSON file storage
